# AMAR HOSPITALS

Multi-page hospital website for AMAR HOSPITALS, a multi-specialty hospital in Ongole, Prakasam District, Andhra Pradesh, India.

## Tech Stack

- React + Vite
- Tailwind CSS
- React Router
- Lucide React (icons)

## Pages

- **Home** — Hero, trust badges, Center of Excellence, Robotic Knee Surgery highlight, 24x7 Emergency banner, CTA
- **About** — Hospital overview, infrastructure, Why Choose Us
- **Services** — Critical Care & Emergency, Orthopedics (Robotic Knee Replacement), Other Specialties
- **Doctors** — Dr. R. Amarnath Reddy profile with placeholder slots for future doctors
- **Contact** — Contact form, appointment call-out, address, embedded map

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
